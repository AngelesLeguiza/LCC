Proyectos Propios

Gestión de Biblioteca

Plataforma web completa para administración de bibliotecas. Permite gestionar inventario de libros, registrar préstamos, controlar devoluciones y generar reportes estadísticos automatizados.

Desarrollé esta solución pensando en hacer más accesible la gestión bibliotecaria, especialmente para instituciones educativas o comunitarias que necesitan organizar sus recursos de manera simple pero efectiva. La interfaz está diseñada para ser intuitiva, permitiendo que cualquier persona pueda usarla sin necesidad de capacitación extensa.

Calculadora Interactiva

Calculadora web funcional con operaciones básicas y diseño moderno. Desarrollada como práctica de lógica de programación y manipulación del DOM.

Este proyecto me permitió entender a fondo cómo capturar interacciones del usuario y procesarlas en tiempo real. Más allá de las operaciones matemáticas, me enfoqué en crear una experiencia fluida con feedback visual inmediato, manejo de errores y una interfaz que responde naturalmente a cada clic.

Gestor de Tareas Personales

Aplicación web para gestionar tareas diarias. Permite agregar, editar, eliminar y marcar tareas como completadas. Los datos se almacenan localmente en el navegador.

Creé esta aplicación para resolver mi propia necesidad de organizar el día a día. Lo más valioso fue implementar el almacenamiento local, lo que significa que tus tareas permanecen guardadas incluso después de cerrar el navegador. Es sorprendente cómo algo tan simple puede hacer una gran diferencia en la productividad personal.

ReservaFácil - Sistema de Reservas

Aplicación web para gestión de reservas en restaurantes. Permite a los clientes reservar mesas online y al personal administrar disponibilidad y horarios.

Desarrollé este sistema pensando en la experiencia tanto del cliente como del restaurante. Los comensales pueden ver en tiempo real qué mesas están disponibles, mientras que el personal tiene una visión clara de la ocupación del día. Fue fascinante resolver el desafío de sincronizar múltiples reservas sin conflictos de horarios.
